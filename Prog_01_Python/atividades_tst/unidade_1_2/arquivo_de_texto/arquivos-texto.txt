Nos arquivos texto os dados são representados linha à linha. Na prática é uma sequência de bytes representando caracteres. As linhas são representadas pelo caracter de quebra de linha, por exemplo: '\n'. Esse caracter vai variar de sistema operacional para sistema operacional. Todos os dados são armazenados como caracteres. por exemplo o número 42 utilizaria um byte para cada dígito do número.

Já os arquivos binários são representados por uma sequência de bytes sem o conceito de quebra de linha. Ele armazena o dado literal, ou seja, não são caracteres. Pode imaginar uma fita sequencial cheia de dados. O número 42 ocuparia apenas 1 byte cujo valor em binário seria 101010.
